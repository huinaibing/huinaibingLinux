### Q8

![image-20250225162032913](./img/image-20250225162032913.png)

由于这个文件夹里面内容比较重要，故移动完后需要创一个链接，否则会出问题

```
# 创建文件夹
sudo mkdir -p /opt/pgms
# 移动
sudo mv /usr/local /opt/pgms/
# 创建链接
sudo ln -s /opt/pgms/local /usr/local
```

命令如下

```
df -k | sed -n "/^\//p"
```

`^\/`表示只匹配行首的`/`，反斜杠是转义字符

```sh
ls -l | 
sed "s/$USER//g" | 
sed "s/ [A-Z][a-z][a-z] //g" | 
sed 's/[0-9]\(.*\):[0-9][0-9]//g'
```

代码首先将当前目录所有东西打印一遍，并传给下一个指令，第二行筛选并且删除用户名，第三行把星期的信息删除，第四行将权限后面的数字到时间这一段全部删除，中间使用正则表达式
